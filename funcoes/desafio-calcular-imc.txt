** ENUNCIADO **
- IMC = peso / altura² - feito
- Deve retornar um único número - feito
- Deve gerar um erro se receber um parâmetro não número - feito
- Deve retornar erro caso não receba nenhum parâmetro - feito

** Função para Classificar o IMC **
- Deve receber um número (IMC) - feito
- Deve retornar uma string
- Deve gerar um erro se receber um parâmetro não número - feito
- Deve retornar erro caso não receba nenhum parâmetro - feito
